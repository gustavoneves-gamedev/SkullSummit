# Skull Summit

Skull Summit is a mobile climbing runner game currently in development, built with Unity and C#.

The project focuses on a stamina-based gameplay system, adding a strategic layer to a fast-paced runner experience.

## Current Features

- lane-based movement
- character selection
- shop/progression
- run summary
- leaderboard
